## 📖 仓库介绍

这是一个完全自动化的 OpenWrt/ImmortalWrt 软件源镜像站。为了解决官方源下载慢、内核版本不匹配（Kernel Mismatch）导致无法安装插件的问题，我建立了这个仓库。

### 主要包含：
* **📂 固件镜像 (Images)**：
    * 包含 `sysupgrade` (升级包)、`factory` (过渡包) 和救砖固件。
* **📦 软件仓库 (Packages)**：
    * 收录了一些常用插件
    * **独家特性**：自动归档不同内核版本的 `kmod` 驱动，确保你刷了任何版本的固件，都能找到匹配的驱动源。

## ⚡ 快速上手

### 1. 下载固件
进入 [网站主页](https://owrt.yaojiu.org/)，点击 **"固件下载"**，选择最新版本刷入路由器。

### 2. 配置软件源
为了能正常安装插件，刷机后请务必更新软件源配置。
在网站的 **"软件仓库"** 或 **"下载页"**，复制自动生成的 `opkg` 配置代码，覆盖路由器 `/etc/opkg/distfeeds.conf` 文件即可。
